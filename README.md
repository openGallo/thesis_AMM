# thesis_AMM

Master thesis project on Automated Market Makers, with emphasis on Uniswap v3, the USDC/WETH 0.05% pool, DEX/CEX price comparison, liquidity provision, and numerical simulations.

## Project structure

- `data_raw/`: raw DEX, CEX, and other data.
- `data_processed/`: cleaned and transformed datasets.
- `scripts/import_data/`: scripts used to collect raw data.
- `scripts/process_data/`: scripts used to clean and transform data.
- `scripts/thesis/`: thesis-specific analysis and simulation scripts.
- `output/tables/`: generated thesis tables.
- `output/figures/`: generated thesis figures.
- `docs/`: data dictionary and dataset manifest.

## Data policy

Large raw and processed datasets are not committed to GitHub.  
The repository stores code, documentation, small samples, and reproducibility instructions.
