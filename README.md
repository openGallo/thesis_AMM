# thesis_AMM вЂ” Public Repository

Empirical study of the USDC/WETH 0.05% pool (`0x88e6A0c2ddd26FEEb64F039a2c41296FcB3f5640`) over January 2022 вЂ“ December 2024, plus a multi-fee-tier extension covering the 0.01%, 0.30%, and 1.00% pools.

> **Note:** Analysis scripts, thesis LaTeX source, and figures live in a **private companion repository** (`thesis-AMM-private`). This public repo contains the data-import, processing pipeline, and integrity checks only.

### Key Results (Main Study Pool вЂ” USDC/WETH 0.05%)

| Quantity | Value |
|---|---|
| Break-even volatility Пѓ\* | 127.7% ann. (П„_d в‰€ 0.975) |
| Reduced-form jump at Пѓ\* | +0.000031 (SE = 0.000024, *p* = 0.20, n.s.) |
| First stage О”P(LVR > fee) | +0.105 (SE = 0.104, F = 1.02, weak) |
| LATE (fuzzy RD / Wald) | +0.000296 (*p* = 0.43, n.s.) |
| Sample | N = 22,368 hourly obs., h_IK = 8.25 pp, N_local = 441 |

Null result is economically interpretable: with П„_d в‰€ 0.975 the Пѓ\* threshold falls in the far right tail of the volatility distribution; only ~10.8% of hours breach it.

## How to run

Open a terminal in the project root. Run in this order:

### 1 вЂ” Import CEX data
```powershell
python scripts/import_data/run_cex.py
```
Downloads Binance klines (1m, 5m) and aggTrades for ETHUSDC, ETHUSDT, USDCUSDT.
**~20вЂ“60 min** depending on connection speed.

### 2 вЂ” Import DEX data (main 0.05% pool)
```powershell
python scripts/import_data/run_dex.py --skip-ticks
```
Fetches pool time series, swaps, mints, burns, collects, and LP positions from The Graph.
**~1вЂ“3 hours** (rate-limited API). Steps 1 and 2 are independent and can run in parallel.

### 3 вЂ” Import DEX data (multi-tier extension: 0.01%, 0.30%, 1.00%)
```powershell
python scripts/import_data/run_multitier_dex.py
```
Downloads hourly/daily pool data for the three additional fee tiers.
**~30вЂ“90 min** per pool. Run individual pools with `--001`, `--030`, `--100`.

### 4 вЂ” Validate all data
```powershell
python scripts/check_data_integrity.py
```
Checks coverage, row counts, OHLCV sanity, temporal gaps, column completeness, and
fee-tier consistency for every dataset. Saves a full report with remediation instructions to
`reports/data_integrity_YYYY-MM-DD.txt`. Exit code 1 if any check fails.

### 5 вЂ” Process everything
```powershell
python scripts/process_data/run_all.py
```
Runs all processing scripts in dependency order. **~2вЂ“5 min.**
